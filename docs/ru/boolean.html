Логическое значение

Булев имеет тип bool и может принимать значения true или false. Основные операции:

  • &&: логическое И
  • ||: логическое ИЛИ
  • !: логическое НЕ.
  • <=, >=, <, >
  • ==: structural equal, compares data structures deeply: (1, 2) == (1, 2) is true. Convenient, but use with caution
  • ===: сравнение по ссылке. (1, 2) === (1, 2) равно false. let myTuple = (1, 2); myTuple === myTuple равно true.
  • !=: structural unequal
  • !==: неравенство по ссылке

Использование

Note: Reason/Bucklescript true and false compile to JS true and false since Bucklescript 3.0. Before that release you had to use Js.to_bool and Js.Boolean.to_js_boolean but thankfully this is no longer the case!

Советы и трюки

Use structural equal tastefully. Это удобно, но вы можете случайно сравнить две глубоко вложенные структуры данных и нанести большой урон производительности. Также не всегда явно, что считается "равным". Например, часть данных foo равна ленивой foo? В идеале это должно быть настраиваемо. В будущем такие изменения могут появится в языке. Если вам интересно, посмотрите этот раздел.
