Regular Expression

Reasonの正規表現はJavaScriptの正規表現に直接変換されます。 そのため、Reasonでは直接JSの正規表現を使うことができます。

```reason let f = [%re "/(0-9)+/g"]; ``` ```js var f = (/(0-9)+/g); ```ご覧下さい、ランタイムコストはありません!
