Integer & Float

整数

32位, 超出截断。Reason为它们提供了常见的操作: +、-、*、/ 等。

用法

请参见标准库中的 Int32 模块 。有关 JS 编译, 请参见 Js.Int 。

提示& 技巧

** 在绑定到 JavaScript 数字时小心 **！长的可能被截断。将 JS 数字绑定为浮点型。

浮点数

Float 需要其他运算符: +., -., *., /., 等等。如 0.5 + 0.6。

用法

当前标准库中没有浮点数模块。有关 js 编译, 请参见 js. Float 。
