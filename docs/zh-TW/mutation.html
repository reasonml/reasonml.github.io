Mutation

Reason 具有好的傳統命令 & 變化的程式能力。 你應該節約地使用這些功能，有時候會讓程式碼更有效率和撰寫成熟悉的樣式。

改變 Let-繫結

Let-繫結是不可變的，但你可以用 ref 包起來，就像是一個盒子，裡面的內容可以改變：

let foo = ref(5);

用法

透過字尾加上 ^ 操作子，就可以取得 ref 實際內容值：

let five = foo^; /* 5 */

賦予新的內容值到 foo 就像：

foo := 6;

注意先前的繫結 five 仍然保持 5，因為取得的是 ref 裡面的內容值，而非 ref 本身。

提示 & 技巧

開玩笑 ref 事實上並不是特殊功能！ 只是個在標準函式庫內的語法糖，其實是預定可變的 record 型別，稱為 ref（請在連結的內容頁面搜尋 "References"）。 下列為原始版本：

let foo = {contents: 5};
let five = foo.contents;
foo.contents = 6;

前往 ref 之前，請知悉你可以透過覆寫輕量、區域 "變化"的 let 繫結達到改變：

let foo = 10;
let foo = someCondition ? foo + 5 : foo;
print_int(foo); /* 結果 15 或 10 */
