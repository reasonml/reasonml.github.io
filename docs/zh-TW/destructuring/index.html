Destructuring

"解構" 是個直覺簡潔的方式，用來取出資料結構的欄位。就像是變數，你可以在任何地方使用解構。

用法

下列綁定變數： ten = 10, twenty = 20

let someInts = (10, 20);
let (ten, twenty) = someInts;

下列綁定變數：name = "Guy", age = 30

type person = {name: string, age: int};
let somePerson = {name: "Guy", age: 30};
let {name, age} = somePerson;

當取出欄位時，可以選擇性重新命名欄位名稱。下列綁定變數：n = "Guy", a = 30.

let {name: n, age: a} = somePerson;

解構允許型別標註。

let (ten: int, twenty: int) = someInts;
let {name: (n: string), age: (a: int)} = somePerson;

函式的標籤式參數也可以解構。

type person = {
  name: string,
  age: int
};

let someFunction = (~person as {name}) => {
  /* 你可以使用 `name` */
};

let otherFunction = (~person as {name} as thePerson) => {
  /* 你可以使用 `name` 和整個 `thePerson` record */
};

繼續閱讀「樣式匹配」章節，有更多解構應用！

提示 & 技巧

解構讓程式碼更加簡潔，不再需要中介變數。盡量使用！但不要濫用造成過度巢狀 & 扼要。

如果解構 record 或變異，若其定義不在同一個檔案，你必須明確標註它。 請見這裡和這裡。
