Boolean

布林的型別為 bool，可以是 true 或者 false。常見操作有：

  • &&：邏輯 and
  • ||：邏輯 or
  • !：邏輯 not
  • <=、>=、<、>
  • ==：結構相等，深層地比對資料結構：(1, 2) == (1, 2) 為 true。方便，但謹慎使用
  • ===：參照相等，淺的比對。 (1, 2) === (1, 2) 為 false。let myTuple = (1, 2); myTuple === myTuple 為 true。
  • !=：結構不相等
  • !==：參照不相等

用法

注意：自 Bucklescript 3.0 開始，Reason/Bucklescript 的 true 跟 false 編譯成 JS true 跟 false。 在這之前，需要使用 Js.to_bool 及 Js.Boolean.to_js_boolean 進行轉換，幸好，現在不再需要了！

提示 & 技巧

優雅地使用結構相等。 這很方便，但可能會意外地讓你比對兩個深層巢狀的資料結構，導致效能低落。 並不是每次都很清楚了解何謂 "相等"。 例如，一塊資料 foo 是否相等緩慢 foo？ 理想上，這是可抽換的。 即將到來的改變將會讓此成真並且可靠；如果有興趣，請見隱含模組化。
